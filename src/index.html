Deformat

Remove extra whitespace characters and line breaks from text.

No more manually removing each line break when pasting large chunks of text from a book. This tool also cleans up any messed up formatting that involves whitespace.

Tips

  • The following characters are removed: \r\n, \r, \n, and extra single space characters.
  • Click the Deformat button to remove whitespace.
  • Use Ctrl+A to select all text.
  • Use the Clear button to remove all text.
